## Installation

Clone the repository:

```bash
git clone https://github.com/yourusername/portfolio.git
```

Navigate to the project folder:

```bash
cd portfolio
```

Open the project:

```bash
index.html
```

Or run using a local development server:

```bash
npx serve
```

## Customization

Update the following files:

### Personal Information
